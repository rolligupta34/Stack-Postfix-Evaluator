# Stack-Postfix-Evaluator (C Program)

This is a console-based Stack Implementation and Postfix Expression Evaluator written in C.  
It allows performing basic stack operations and evaluating postfix expressions using a stack.

## Features Included
- Push, Pop, Peek, Display  
- Postfix Expression Evaluation  
- Error handling for invalid expressions  
- Menu-driven interaction  

## Files in This Project
- stack_postfix.c — Main source code  
- README.md — Project description  


## Concepts Used
- Arrays  
- Structures  
- Stack Operations  
- Postfix Evaluation
